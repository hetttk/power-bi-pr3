# Power BI DAX & Data Modeling Project

## Overview

This project focuses on implementing DAX calculations, calculated columns, measures, and time intelligence functions in Power BI using Sales and Returns datasets. Matrix visuals were used to validate and analyze all calculations.

## Features Implemented

* Created calculated columns:

  * Profit
  * Return Flag
  * Customer Full Name
* Built DAX measures:

  * Total Sales
  * Total Cost
  * Total Profit
  * Return Rate
  * Average Sale per Transaction
* Used Quick Measures for:

  * Year-over-Year Growth
  * Previous Month Comparison
* Created dedicated Measure Table
* Applied filter context using:

  * ALL()
  * FILTER()
  * CALCULATE()
* Used DAX functions:

  * SUM, AVERAGE, COUNTX, DISTINCTCOUNT
  * IF, SWITCH
  * RELATED
  * SUMX, AVERAGEX
  * TOTALYTD, SAMEPERIODLASTYEAR

## Analysis Performed

* Sales by Region
* Revenue by Customer Segment
* Monthly & Yearly Sales Trends
* Return analysis using time intelligence

## Deliverables

* Power BI (.pbix) file
* DAX measures & calculated columns
* Matrix-based analytical reports
* Project documentation

## Conclusion

This project demonstrates practical Power BI DAX concepts including calculated columns, measures, filter context, relationships, and time intelligence for advanced business analysis.
